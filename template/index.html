About Godos

Godos is a minimalist todo list application crafted in Golang. It leverages HTMX for enhanced interactivity and SQLite for seamless data persistence.

Developed with the Echo web framework, Godos employs standard HTML to dynamically render data-driven markup, ensuring a minimalist and visually appealing aesthetic.
